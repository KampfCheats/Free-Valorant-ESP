## Getting Started

### Prerequisites

You need to have Windows 10 installed (Win 11 not compatible)



### Installation

1. Download all the Files
2. Unpack the files into a folder
3. Go into the folder with cmd and run
   ```sh
   Mapper.exe Kernel.sys
   ```
4. Run Valorant.exe
